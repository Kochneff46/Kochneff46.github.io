# AutoGatesPro static site

Статический сайт AutoGatesPro для бесплатной публикации без Tilda.

## Визуальный стиль

Сайт использует фирменные HTML/CSS-иллюстрации вместо фото-заглушек:

- главный рекламный экран с откатными воротами, приводом, шлагбаумом, пультом, сигнальной лампой, искрами и оранжевыми мазками;
- темные карточки услуг с мини-иконками;
- техническая панель диагностики поломок;
- премиальный блок брендов автоматики;
- стилизованная темная карта Москвы и области;
- сервисная зона ТО с инструментами, чек-листом и диагностической консолью.

Файлы `hero.webp`, `services.webp`, `brands.webp`, `maintenance.webp`, `locations.webp` больше не используются сайтом.

## Структура

- `index.html` — главная страница с SEO, OpenGraph и JSON-LD LocalBusiness.
- `styles.css` — премиальный адаптивный темный дизайн, CSS-иллюстрации, CTA, hover-анимации.
- `script.js` — плавная прокрутка, текущий год и появление блоков.
- `robots.txt` — правила индексации.
- `sitemap.xml` — карта сайта.
- `assets/images/` — папка оставлена для будущих настоящих фото или OG-изображения.

## Бесплатная публикация на Netlify

1. Откройте [netlify.com](https://www.netlify.com/).
2. Войдите в аккаунт или зарегистрируйтесь.
3. Перейдите в `Sites`.
4. Нажмите `Add new site`.
5. Выберите `Deploy manually`.
6. Перетащите папку `autogatespro_site` в область загрузки.
7. Дождитесь публикации.
8. Для бесплатного адреса используйте домен Netlify, например `https://kochneff46.github.io`.

## Перед публикацией

1. Проверьте кнопки звонка, WhatsApp, Telegram и Email.
2. Проверьте социальные кнопки Instagram, YouTube, Дзен и VK.
3. Откройте сайт на телефоне и убедитесь, что нет горизонтального скролла.
4. Если добавите настоящее OG-изображение, подключите его в OpenGraph и JSON-LD.
